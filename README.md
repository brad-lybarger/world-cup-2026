# World Cup 2026 — Knockout Bracket

A native Android app focused on the **knockout rounds** of the 2026 FIFA World Cup.

## Features

1. **Weighted random simulator** — Starting from the current standings/bracket, simulate
   each remaining knockout match. The outcome is random but weighted by each team's FIFA
   ranking, so higher-ranked teams are more likely (but not guaranteed) to advance.

2. **Bracket prediction game** — Fill in your own predictions and earn points for each
   correct pick. Scoring doubles each round:

   | Round            | Points per correct pick |
   |------------------|-------------------------|
   | Round of 32      | 1                       |
   | Round of 16      | 2                       |
   | Quarterfinals    | 4                       |
   | Semifinals       | 8                       |
   | Final            | 16                      |

## Tech stack

- **Kotlin** + **Jetpack Compose**
- Android Studio / Gradle
